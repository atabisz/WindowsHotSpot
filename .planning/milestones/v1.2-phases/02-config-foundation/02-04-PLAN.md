Rewrite WindowsHotSpot/Core/CornerDetector.cs. The dwell detection state machine logic (Idle/Dwelling/Triggered, drag suppression, re-arm) must remain EXACTLY unchanged. Only the constructor, UpdateSettings, and OnDwellComplete need to change.

**Changes:**

1. **Replace constructor parameters**: The constructor currently takes `(HotCorner corner, int zoneSize, int dwellDelay)`. Replace with `(HotCorner corner, int zoneSize, int dwellDelay, ConfigManager configManager)`. Store `_configManager` for the CornerActions lookup at dispatch time. All other fields remain the same.

   ```csharp
   private HotCorner _activeCorner;
   private int _zoneSize;
   private int _dwellDelay;
   private readonly ConfigManager _configManager;   // for CornerActions lookup only

   public CornerDetector(HotCorner corner, int zoneSize, int dwellDelay, ConfigManager configManager)
   {
       _activeCorner = corner;
       _zoneSize = zoneSize;
       _dwellDelay = dwellDelay;
       _configManager = configManager;

       _dwellTimer = new System.Windows.Forms.Timer { Interval = _dwellDelay };
       _dwellTimer.Tick += OnDwellComplete;
   }
   ```

2. **UpdateSettings**: Signature stays at 3 params (HotCorner, int, int) — unchanged:
   ```csharp
   public void UpdateSettings(HotCorner corner, int zoneSize, int dwellDelay)
   {
       _activeCorner = corner;
       _zoneSize = zoneSize;
       _dwellDelay = dwellDelay;
       _dwellTimer.Interval = dwellDelay;
       _dwellTimer.Stop();
       _state = DetectorState.Idle;
   }
   ```

3. **OnDwellComplete**: Replace the `ActionTrigger.SendTaskView()` call with ActionDispatcher:
   ```csharp
   private void OnDwellComplete(object? sender, EventArgs e)
   {
       _dwellTimer.Stop();
       CornerAction action = _configManager.Settings.CornerActions[_activeCorner];
       ActionDispatcher.Dispatch(action);
       _state = DetectorState.Triggered;
   }
   ```

4. **Add using directives**:
   - `using WindowsHotSpot.Config;` (already present)
   - No new usings needed — ActionDispatcher is in same namespace (WindowsHotSpot.Core)

The complete rewrite should preserve all of: OnMouseMoved, OnMouseButtonChanged, IsInAnyCornerZone, GetCornerPoint, Dispose — exactly as they are. Only constructor, UpdateSettings, OnDwellComplete, and field declarations change.
