**HotSpotApplicationContext.cs** (namespace WindowsHotSpot):
- Inherits `ApplicationContext`
- Private fields: `_hookManager` (HookManager), `_cornerDetector` (CornerDetector), `_trayIcon` (NotifyIcon), `_contextMenu` (ContextMenuStrip)
- Constructor:
  1. Create ContextMenuStrip with three items:
     - "Settings" -- click handler shows `MessageBox.Show("Settings will be available in a future update.", "Settings", MessageBoxButtons.OK, MessageBoxIcon.Information)` (TRAY-04, placeholder for Phase 2)
     - "About" -- click handler shows `MessageBox.Show("WindowsHotSpot v0.1.0\n\nMacOS-style hot corners for Windows.\nMove your mouse to a screen corner to trigger Task View.", "About WindowsHotSpot", MessageBoxButtons.OK, MessageBoxIcon.Information)` (TRAY-05)
     - Separator
     - "Quit" -- click handler calls `ExitThread()` (TRAY-06)
  2. Create NotifyIcon (TRAY-02):
     - Icon = embedded app.ico resource OR SystemIcons.Application as fallback
     - Text = "WindowsHotSpot"
     - ContextMenuStrip = the menu created above
     - Visible = true
  3. Create CornerDetector (from Plan 01)
  4. Create HookManager (from Plan 01)
  5. Wire events: `_hookManager.MouseMoved += _cornerDetector.OnMouseMoved`
  6. Wire events: `_hookManager.MouseButtonChanged += _cornerDetector.OnMouseButtonChanged`
  7. Install the hook: `_hookManager.Install()`
  8. Wire Application.ApplicationExit to cleanup handler (belt-and-suspenders with Dispose)

- Override `Dispose(bool disposing)`:
  1. If disposing: dispose _hookManager, _cornerDetector, set _trayIcon.Visible = false, dispose _trayIcon, dispose _contextMenu
  2. Call base.Dispose(disposing)
