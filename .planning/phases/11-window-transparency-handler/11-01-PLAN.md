   ```csharp
   _lCtrlDown  = _lCtrlDown  && IsPhysicallyDown((int)NativeMethods.VK_LCONTROL);
   _lAltDown   = _lAltDown   && IsPhysicallyDown((int)NativeMethods.VK_LMENU);
   _lShiftDown = _lShiftDown && IsPhysicallyDown((int)NativeMethods.VK_LSHIFT);
   ```

   code (GetWindowRect, w/h, fx/fy, screen clamping, SM_CXMINTRACK, SetWindowPos, SWP_FLAGS). Replace
   with:
   ```csharp
   // Read current extended style
   int exStyle = NativeMethods.GetWindowLong(rootHwnd, NativeMethods.GWL_EXSTYLE);

   // Read existing alpha and flags to preserve LWA_COLORKEY on color-key windows — TRNSP-03
   // GetLayeredWindowAttributes only succeeds if WS_EX_LAYERED was set via SetLayeredWindowAttributes
   // (not UpdateLayeredWindow). On failure use defaults: alpha=255 (opaque), no flags.
   byte existingAlpha = 255;
   uint existingFlags = 0;
   if ((exStyle & (int)NativeMethods.WS_EX_LAYERED) != 0)
       NativeMethods.GetLayeredWindowAttributes(
           rootHwnd, out _, out existingAlpha, out existingFlags);

   // Compute new alpha — TRNSP-02 (scroll up = more opaque), TRNSP-05 (clamp 25-255)
   int step      = delta * _settings.TransparencyStep / 120;
   byte newAlpha = (byte)Math.Clamp(existingAlpha + step, 25, 255);

   // Ensure WS_EX_LAYERED is set before calling SetLayeredWindowAttributes
   if ((exStyle & (int)NativeMethods.WS_EX_LAYERED) == 0)
       NativeMethods.SetWindowLongPtr(
           rootHwnd,
           NativeMethods.GWL_EXSTYLE,
           new IntPtr(exStyle | (int)NativeMethods.WS_EX_LAYERED));

   // Apply alpha; OR in LWA_ALPHA so it is always active alongside any existing color-key flag — TRNSP-03
   NativeMethods.SetLayeredWindowAttributes(
       rootHwnd, 0, newAlpha, existingFlags | NativeMethods.LWA_ALPHA);
   ```

8. **KeyboardCallback switch** — add VK_LSHIFT case after the VK_LMENU case (NO `!isInjected` guard):
   ```csharp
   case NativeMethods.VK_LSHIFT:
       if (isKeyDown) _lShiftDown = true;   // no !isInjected guard — AltGr does not synthesize LShift
       if (isKeyUp)   _lShiftDown = false;
       break;
   ```

9. **Dispose()** — add `_lShiftDown = false;` after `_lAltDown = false;`.

## Trust Boundaries

| Boundary | Description |
|----------|-------------|
| WH_MOUSE_LL → OnMouseWheeled | Untrusted scroll delta and cursor position from the global hook |
| WindowTransparencyHandler → foreign process window | SetWindowLongPtr and SetLayeredWindowAttributes cross process boundaries |

## STRIDE Threat Register

| Threat ID | Category | Component | Disposition | Mitigation Plan |
|-----------|----------|-----------|-------------|-----------------|
| T-11-01 | Tampering | SetLayeredWindowAttributes on elevated window | mitigate | IsElevatedProcess guard returns early before any write; UIPI enforces at OS level (GUARD-03) |
| T-11-02 | Tampering | Alpha driven to 0 (invisible window) | mitigate | Math.Clamp(alpha, 25, 255) in OnMouseWheeled — minimum enforced in code (TRNSP-05) |
| T-11-03 | Spoofing | AltGr (RCtrl+LAlt) triggering gesture | mitigate | LLKHF_INJECTED guard on VK_LCONTROL case in KeyboardCallback — AltGr's synthesized LCtrl is always injected (GUARD-01) |
| T-11-04 | Tampering | Modifier state stuck after Secure Attention Sequence | mitigate | Self-heal pattern: each modifier AND'd with IsPhysicallyDown() on every OnMouseWheeled call |
| T-11-05 | Denial of Service | Applying transparency to maximized window (breaks restore) | mitigate | GetWindowPlacement + SW_SHOWMAXIMIZED check returns early before any write (GUARD-02) |
| T-11-06 | Tampering | Destroying LWA_COLORKEY on color-key window | mitigate | GetLayeredWindowAttributes reads existing dwFlags; new flags = existingFlags OR LWA_ALPHA (TRNSP-03) |
</threat_model>
