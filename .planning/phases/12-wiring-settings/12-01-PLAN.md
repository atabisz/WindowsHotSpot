## Trust Boundaries

| Boundary | Description |
|----------|-------------|
| SettingsForm → HotSpotApplicationContext | Form returns user-supplied int values via public properties; context reads and assigns to settings |
| settings.TransparencyStep → WindowTransparencyHandler | Integer step value flows from settings into alpha arithmetic at wheel-event time |

## STRIDE Threat Register

| Threat ID | Category | Component | Disposition | Mitigation Plan |
|-----------|----------|-----------|-------------|-----------------|
| T-12-01 | Tampering | SettingsForm._transparencyStepInput | accept | NumericUpDown Minimum=1/Maximum=50 enforces range at the control level; `Math.Clamp` on initial value prevents out-of-range load from persisted settings |
| T-12-02 | Denial of Service | WheelSuppressionPredicate OR lambda | accept | Both delegates are local method calls with no I/O; lambda executes in <1µs; no DoS surface |
| T-12-03 | Information Disclosure | DisposeComponents unsubscribe ordering | mitigate | Both MouseWheeled handlers unsubscribed before WheelSuppressionPredicate = null; prevents stale lambda holding live handler references after dispose |
</threat_model>
